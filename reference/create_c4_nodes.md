# Create C4 nodes

Create C4 nodes

## Usage

``` r
create_c4_nodes(person, system_or_container, external_system, level)
```

## Arguments

- person:

  List of person elements

- system_or_container:

  List of system/container/component elements

- external_system:

  List of external system elements

- level:

  Character string indicating diagram level

## Value

Data frame of nodes
