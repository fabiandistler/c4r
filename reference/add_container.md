# Add a container element to the builder

Add a container element to the builder

## Usage

``` r
add_container(builder, id, label = id, description = "", technology = "")
```

## Arguments

- builder:

  A c4_builder object

- id:

  Element identifier

- label:

  Element label

- description:

  Element description

- technology:

  Technology stack

## Value

Updated builder object
