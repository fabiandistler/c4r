# c4r: Create C4 Architecture Diagrams

Provides functions to create C4 architecture diagrams (Context,
Container, Component, and Code diagrams) using a simple R interface. The
package generates diagrams in DOT notation that can be rendered using
DiagrammeR or other Graphviz-compatible tools.
