# Add a group/boundary to the builder

Add a group/boundary to the builder

## Usage

``` r
add_group(builder, id, label = id, members = character(0), style = "dashed")
```

## Arguments

- builder:

  A c4_builder object

- id:

  Group identifier

- label:

  Group label

- members:

  Element IDs in this group

- style:

  Border style

## Value

Updated builder object
