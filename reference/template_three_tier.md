# Three-tier architecture template

Three-tier architecture template

## Usage

``` r
template_three_tier(
  title = "Three-Tier Architecture",
  web_tech = "React",
  app_tech = "Node.js",
  db_tech = "PostgreSQL"
)
```

## Arguments

- title:

  Diagram title

- web_tech:

  Web technology

- app_tech:

  Application technology

- db_tech:

  Database technology
