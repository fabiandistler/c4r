# Generate DOT notation for C4 diagram

Generate DOT notation for C4 diagram

## Usage

``` r
generate_c4_dot(title, nodes, edges, theme, level, groups = list())
```

## Arguments

- title:

  Character string for diagram title

- nodes:

  Data frame of nodes

- edges:

  Data frame of edges

- theme:

  Character string for theme or c4_theme object

- level:

  Character string for diagram level

- groups:

  List of group objects (optional)

## Value

Character string with DOT notation
