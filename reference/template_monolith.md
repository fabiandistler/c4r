# Monolith template

Monolith template

## Usage

``` r
template_monolith(
  title = "Monolithic Architecture",
  app_name = "Application",
  database = "postgres",
  cache = NULL
)
```

## Arguments

- title:

  Diagram title

- app_name:

  Application name

- database:

  Database type

- cache:

  Cache type (optional)
